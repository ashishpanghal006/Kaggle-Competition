# Kaggle Competitions

### 1. Cinema Audience Forecasting – Machine Learning Project

This project was completed as part of the MLP Course – IITM BS Degree Program.
The goal of the problem was to forecast daily audience count for movie theaters using historical booking and visit patterns.
The dataset was provided as part of a private Kaggle competition, where submissions were evaluated based on prediction accuracy.

Three models were trained: 
- LightGBM Regressor
- XGBoost Regressor
- Random Forest Regressor
